# Features

## Framework agnostic

Use it with Jest, Vitest, Bun, Node test runner, or other ecosystems.

## Lazy mock creation

Accessing a property creates only what you need, reducing manual setup.

## Deep mocking support

Model nested structures while keeping strong TypeScript inference.

## Strict behavior options

Use strict variants to fail fast when tests access unconfigured behavior.
