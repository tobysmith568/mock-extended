# mock-extended

Bun + TypeScript monorepo scaffold using Bun workspaces.

## Structure

- apps/ — application projects (e.g. website)
- packages/ — reusable/publishable packages

## Getting started

Install dependencies:

```bash
bun install
```

Run checks:

```bash
bun run check
bun run typecheck
```

Format code:

```bash
bun run format
```
